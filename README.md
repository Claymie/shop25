# 🐾 MOPSIK SHOP — Archive Fashion Concept Store

Стильный концепт-магазин архивной и дизайнерской одежды, созданный на чистом коде. Проект демонстрирует современные подходы к веб-верстке и визуальному интерфейсу в эстетике high-end брендов (Rick Owens, Balenciaga, Chrome Hearts).

## 🚀 Фишки проекта

* **Premium UI**: Глубокий темный интерфейс с акцентом на типографику и контент, передающий атмосферу "архивной" моды.
* **Smart Catalog**: Динамическая система фильтрации вещей по брендам, работающая мгновенно без перезагрузки страниц.
* **Responsive Layout**: Использование адаптивных сеток, благодаря которым магазин выглядит одинаково круто как на широком мониторе, так и на экране смартфона.
* **Interactive UX**: Кастомный курсор и плавные анимации навигации, реализованные с помощью чистого JavaScript.
* **Utility Features**: Интерактивная таблица размеров и форма обратной связи для оформления заказов.

## 🛠 Техническая база

* **HTML5**: Грамотная структура документа и использование семантических тегов.
* **CSS3**: Сложная верстка на **CSS Grid** и **Flexbox**, кастомные стили и `@media` запросы для полной адаптивности.
* **JavaScript (ES6+)**: Логика фильтров, управление состоянием интерфейса и обработка пользовательских событий.
* **GitHub Pages**: Использование профессионального хостинга для публикации проекта в открытом доступе.

## 📦 Файловая структура

* `index.html` — Главная страница, репрезентация бренда и форма захвата заявок.
* `catalog.html` — Функциональный каталог с карточками товаров и системой фильтров.

## 💡 Цель проекта
Показать, что для создания эстетичного и быстрого e-commerce продукта не нужны конструкторы. Только чистый код, внимание к деталям и понимание целевой аудитории.
